## Key Concepts

### Monorepo Architecture
AVRO uses a monorepo structure to manage 100+ interconnected projects with standardized patterns and shared tooling.

### Task Registry
Centralized task management system tracking 5K+ structured tasks with dependencies, metadata, and AI execution context.

### AI-First Design
The platform is optimized for AI agent workflows with context files, autonomy rules, and automated issue-to-PR workflows.

### Human Oversight
While enabling autonomous operations, critical decisions and supervised changes require human review.
